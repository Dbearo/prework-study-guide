Prework Study Guide

HTML

  • the head element contains info about the webpage that users don't see
  • the body element contains info that the users see
  • the meta element contains info for the browser to work
      • different elements for the body include:
    • h1-h6 (for headers)
    • ul and ol (for lists) and li (for list items)
    • p (for paragraphs or general text)
    • img (for images)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • you can use hexadecimal code to choose colors, or just use the words themselves for basic colors
  • use text-align to choose where the assets go or use padding-left, right etc.
  • use external css to apply the changes over the whole html file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
